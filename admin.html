📝 How to Change Markup

app.py এর শুরুতে এই variables change করো:

MARKUP_MODE = "percent" # বা "fixed"
MARKUP_AMOUNT = 100 # 100% মানে double price
MIN_PRICE_BDT = 25 # সর্বনিম্ন ৳25
USD_TO_BDT = 110 # exchange rate Change করার পর Render.com এ redeploy হবে automatically।
